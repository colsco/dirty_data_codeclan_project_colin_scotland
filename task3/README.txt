CodeClan Dirty Data 28 Mar 2022

Sea Bird Observation Data

This task has the following file structure;

> clean_data --> contains 'birds_clean.csv'; a cleaned version of 'seabirds.csv'

> data_cleaning_scripts --> R script for data cleaning

> documentation_and_analysis --> bird_count.html analysis report

> raw_data --> contains 'seabirds.csv'; the original raw data set


The project file 'task3.Rmd' is in the project top level folder.  All scripts can be run from here.
